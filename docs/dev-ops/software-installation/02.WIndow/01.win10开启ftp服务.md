# win10 开启 ftp 服务

## 一、启动ftp服务

控制面板 -> 程序和功能 -> 启动或关闭 Windows 功能。

![1](https://juzicoding.com/img/2025/01KDAEBTS44NRYD6KH8ZPGRE7B.webp)

如上图勾选，点击确定，搜索iis打开iis。

![2](https://juzicoding.com/img/2025/01KDAEBWY6HY9YSDZYBR8JCA56.webp)

右击网站 -> 添加 FTP 站点 -> 输入FTP 站点名称和 FTP 目录。

![3](https://juzicoding.com/img/2025/01KDAEBZFXZPF36B52RS2DZMCX.webp)

IP 地址填 FTP 本机 IP；选择 无SSL。

![4](https://juzicoding.com/img/2025/01KDAEC1KSXHQ2979F1NGVWXHG.webp)

根据需求设置身份验证和授权，设置成所有人可以读取，无法写入，暂时先这样设置。

![5](https://juzicoding.com/img/2025/01KDAEC3PT8TA5V30JQ0ZABG9A.webp)

控制面板 -> Windows Defender 防火墙 -> 允许应用或功能通过Window Defender 防火墙 -> 更改设置 -> 勾选 FTP 服务器专用和公用。

![6_1](https://juzicoding.com/img/2025/01KDAEC5KZGRBRHY24FBWYVNJZ.webp)

![6_2](https://juzicoding.com/img/2025/01KDAEC7QY5KZKVSDT74NXY4EP.webp)

到了这一步，ftp 服务就已经可以使用了，使用 `ftp://ip地址` 即可访问。

## 二、设置需要权限登录

创建登录用户：右击此电脑 -> 管理 -> 右击用户 -> 新用户。

![7](https://juzicoding.com/img/2025/01KDAECA1GQC3GM45KV6SHW65D.webp)

输入自己想要的用户名称和密码，创建用户用于 ftp 登录验证。

点击创建好的 ftp 站点 -> 点击 FTP 身份验证 -> 右击禁用匿名身份验证。

![8_1](https://juzicoding.com/img/2025/01KDAECCAGS7GNEMKR3MRHBX2W.webp)

![8_2](https://juzicoding.com/img/2025/01KDAECEFTKCM0YS5C2J4H2SKA.webp)

点击 FTP 授权规则 -> 编辑 FTP 授权规则 -> 填入自己创建的用户名即可。

![9_1](https://juzicoding.com/img/2025/01KDAECGQVZEMW4T5N4K0N93HF.webp)

![9_2](https://juzicoding.com/img/2025/01KDAECK7RQX3RGBST79Y0ZZF0.webp)

尝试访问，发现已经生效。

![10](https://juzicoding.com/img/2025/01KDAECN8VY4QH1TTYRGEW8HEN.webp)
